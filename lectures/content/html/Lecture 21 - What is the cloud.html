What is the cloud?

And what is cloud computing?

How would you define it?

PC Mag says this:

Cloud: A communications network. The word "cloud" often refers to the Internet, and more precisely to some data center full of servers that is connected to the Internet.

Wikipedia says this:

Cloud computing is a type of Internet-based computing that provides shared computer processing resources and data to computers and other devices on demand.

Webopedia says this:

Cloud computing is a type of computing that relies on sharing computing resources rather than having local services or personal devices to handle applications

So in essence...

  • "The cloud" really just means a bunch of servers
  • That are connected to the Internet
  • That provide a service over the Internet
  • Whose resources can be shared to accomplish a purpose

Let's imagine

  • We all have computers
  • Each of our computers have resources that can be used to run an application
    • CPU: processing power
    • RAM: fast, short term storage
    • Storage: slower long term storage

To run an application

  • We would have to ensure that our one computer has enough resources to load the application
  • And enough resources to run the application smoothly
  • And enough resources to be able to accept user input
  • And the application has to run in a reasonable amount of time

What if we can't do that?

What can we do?

What if we can't do that?

If we can communicate between computers, maybe we can use more than one computer to run the application

Perhaps we can share resources?

Example Climateprediction.net

  • climateprediction.net
  • Purpose is to use computers all over the web to run climate models
  • Uses spare CPU/RAM resources on your computer to compute tasks
  • Receive instructions and send results through the Internet
  • Your computer is part of the cloud!

Example Climateprediction.net

  • By using people's personal computers, they can compute climate models that would take supercomputers time to complete
  • The Internet provides a communication path for networked computers to share information

Example Climateprediction.net

  • Uses a client program that is installed on the computer
  • Client program uses the Internet to communicate to the host servers

Example Dropbox

  • File sharing/syncing service
  • Provides seamless sharing of files
  • How does it work?

Dropbox Client

  • Installed on your computer
  • Detects changes in files from your computer and sends those changes through the Internet to Dropbox's "file system" servers
  • Downloads changes made from other sources through the Internet

Dropbox Web site / Mobile application

  • Available through the browser
  • Dropbox web servers interface with Dropbox "file system" servers to learn what files you have in the cloud
  • Provides access through HTTP traffic to your Dropbox files

Dropbox

  • Multiple servers solve the storage problem
  • Can store more data than your single computer can handle
  • Can store more data than your mobile device can store
  • Can provide access to files to devices you don't own through the website

Hosting in cloud

By applications being hosted in the cloud, it releases the need to be bound by physical hardware
